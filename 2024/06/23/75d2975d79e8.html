Endnote中修改引用文献的格式和目录格式

Posted on 2024-06-23 In Share knowledges Views: Symbols count in article: 989

Endnote 可以自动对文献进行管理，是一个非常好用的文献管理数据库，结合 Microsoft Word 软件，可以实现对参考文献进行自动引用和插入，这其中比较重要的是如何自定义参考文献的引用格式和输出的目录格式，下面将详细介绍。

1. 新建输出样式

Endnote 中自定义了非常多的输出样式，并且还支持自定义样式，下面介绍如何自定义输出样式。

对于老版本的 Endnote, 找到 Edit -> Output Styles, 对于 Endnote 20, 在 Tools -> Output Styles 下

如下图所示：

其中 New Style 是创建新的引用和输出格式，另外 Endnote 本身也内置了许多的输出格式，我们选择一个已有的格式作为模板，来进一步修改。

比如这里可以在 Style Manager 中找到找到 Science style，然后点击 Edit，如下图：

2. 修改引用格式

打开 Science style 的编辑界面之后，找到 Citations -> Templates，在 Citation 文本框中可以编辑引用文字格式，如下图：

为了不破坏原来的 Style，这里建议先保存一个副本，然后在副本的基础上修改，操作为 file -> Save As -> 设置 Style name。

这里我们把原模板重新保存为 Science-new1，然后打开模板，修改引用格式，下面介绍如何修改：

例如这里我们把引用格式设置为斜体加粗方括号上标，如下图：

然后 Ctrl+s 快捷键保存修改，打开 Word，光标选中要插入文献的位置，更改 Style 为我们刚刚创建的 Science-new1, 按下 Alt+2 快捷键，即可快速生成引用和文献目录，如下图：

可以看到引用格式变成了我们指定的格式。

3. 修改目录格式

修改样式在 Bibliography -> Templates 下，如下图：
